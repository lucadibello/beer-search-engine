## 1.1. Getting started

### 1.1.1. Create project environment

```bash
conda env create -f environment.yml
```

### 1.1.2. Activate environment

```bash
conda activate beer-search-engine
```

### 1.1.3. Install requirements

```bash
pip install -r requirements.txt
```
